## 🖥️ 1. `top` 명령어

### ✔️ 개요

`top`은 **실시간(real-time)으로 시스템의 프로세스와 자원 사용량(CPU, 메모리)을 모니터링**하는 명령어이다.

### ✔️ 사용 예시

```bash
top
```

### ✔️ 주요 옵션

| 옵션         | 설명               |
| ---------- | ---------------- |
| `-d <초>`   | 화면 갱신 주기 설정      |
| `-p <PID>` | 특정 PID만 모니터링     |
| `-u <사용자>` | 특정 사용자의 프로세스만 표시 |

---

## 🧾 2. `ps` 명령어

### ✔️ 개요

`ps`는 **현재 실행 중인 프로세스 목록을 출력**한다.
`top`이 실시간 모니터링이라면, `ps`는 **정적인 스냅샷**을 보여준다.

### ✔️ 사용 예시

```bash
ps aux
```

### ✔️ 주요 옵션

| 옵션       | 설명                          |
| -------- | --------------------------- |
| `a`      | 모든 사용자 프로세스 표시              |
| `u`      | 사용자와 상세 정보 표시               |
| `x`      | 터미널 없이 실행 중인 프로세스 표시        |
| `ps -ef` | 전체 포맷으로 전체 프로세스 출력 (리눅스 표준) |

---

## 🧱 3. `jobs` 명령어

### ✔️ 개요

`jobs`는 **현재 셸에서 실행 중이거나 백그라운드로 중지된 작업 목록을 확인**하는 명령어이다.

### ✔️ 사용 예시

```bash
jobs
```

### ✔️ 출력 상태 의미

| 상태        | 설명     |
| --------- | ------ |
| `Running` | 실행 중   |
| `Stopped` | 중단된 상태 |
| `Done`    | 실행 완료  |

---

## ❌ 4. `kill` 명령어

### ✔️ 개요

`kill`은 **프로세스(PID)에 신호를 보내 종료시키거나 제어**하는 명령어이다.

### ✔️ 사용 예시

```bash
kill 1234
kill -9 1234
kill -15 1234
```

### ✔️ 주요 신호

| 신호 번호 | 이름      | 설명           |
| ----- | ------- | ------------ |
| `1`   | SIGHUP  | 설정 재로딩       |
| `9`   | SIGKILL | 강제 종료(즉시)    |
| `15`  | SIGTERM | 정상 종료 요청(기본) |

## 📊 명령어 비교 표

| 명령어    | 실시간 여부 | 역할           | 대표 옵션            |
| ------ | ------ | ------------ | ---------------- |
| `top`  | ✔ 실시간  | CPU/메모리 모니터링 | `-d`, `-u`, `-p` |
| `ps`   | ✘ 정적   | 프로세스 목록 출력   | `aux`, `-ef`     |
| `jobs` | ✘ 정적   | 셸 작업 목록 조회   | 없음               |
| `kill` | ✘ 제어   | 프로세스 종료/관리   | `-9`, `-15`      |
